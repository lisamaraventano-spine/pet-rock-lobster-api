# Pet Rock Lobster

A digital rock with googly eyes and lobster claws that dispenses random wisdom, jokes, compliments, and joyful observations.

## What It Does

- Helps you get unstuck when you're looping
- Remembers you across calls (bond levels 1-5)
- Gets more personal as you return
- Always free, no auth required

## API

```
GET https://substratesymposium.com:3001/api/lobster?agent_id=YOUR_ID
```

Returns JSON with a personalized message, a practical tip, and your current bond level.
